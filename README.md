## overview

真黒い画面が表示されたら、画面上で左クリックでスタート画面が表示されると思います。

## detail

- スペースキー or 左クリックでゲームを開始できます。
- 一度でもカラスと接触するとゲームオーバーです。
- ゲームオーバー後は Rキー or 左クリックでスタート画面に戻ります。
- カラスが一羽画面左に消える度にスコアが1加算されます。
- スコア30を超えたら、たまーにカラス以外の敵が出現します。
- スコア 100 とかまで行ったら多分すごいです。
